The project stands as a digital study of elegance, proving that educational data can be presented through a premium, immersive sensory experience.

🛠️ The Tech Garden (Stack)

🖋️ TypeScript: Pure architectural integrity with robust type-safety for a seamless developer experience.
⚛️ React 19: Utilizing the latest component patterns for high-speed, interactive user interfaces.
🎨 Tailwind CSS 4.0: Leveraging next-gen utility styling and dynamic @theme variables for a cohesive visual system.
🎞️ Motion: Powering fluid, scroll-triggered transitions and elegant background color shifts that mirror nature's flow.
🚀 Vite 6: The engine behind the scenes, ensuring lightning-fast performance and modern bundling.
🫧 Lucide Icons: Minimalist, feather-light iconography that complements the floral theme.
 
✨ Signature Features

🌈 Dynamic Atmosphere: The entire application’s background and accent colors morph in real-time as you scroll through different color sections.
📖 Curated Herbarium: A deep-dive catalog of flowers ranging from the common to the exotic, complete with historical "Botanical Notes."
🏔️ Intuitive Navigation: A fixed, glass-morphism header allowing one-click jumps across the entire color spectrum.
📱 Aesthetic Grid: A responsive "Bento-style" layout that adapts beautifully from large displays to mobile devices.
